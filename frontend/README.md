# Проект: Место-реакт
---------
Этот проект рассказывает Вам делиться с окружающими своими фотографиями в путешествиях.

**Используемые технологии:**
* HTML
* CSS
* JavaScript
* Nested БЭМ
* React
* Webpack

Для сборки и запуска проекта используется команда npm start.
